StaySummary

Privacy Policy

Introduction

StaySummary is a Chrome extension that uses AI to summarize accommodation reviews across major booking platforms including Airbnb, Booking.com, Vrbo, Expedia, Hotels.com, TripAdvisor, and Agoda. We are committed to protecting your privacy.

This policy explains what data the extension accesses, how it is processed, and which third-party services are involved.

In short: We do not collect, store, or sell any personal information. Review and location data from listing pages is sent to AI and mapping services solely to generate summaries — it is not stored on our servers.

What Data the Extension Accesses

When you click “Summarize” on a supported listing page, the extension reads the following information directly from that page:

  • Listing reviews — Guest review text visible on the page (or fetched via the platform’s public API for Airbnb).
  • Listing title and nightly price — Used to provide context in the AI summary.
  • Listing coordinates (latitude/longitude) — Extracted from page metadata or map links to enable neighborhood and safety analysis.

None of this data is tied to your identity. The extension does not access your account on any booking platform, your search history, or any personal information.

What Data Is Stored Locally

The extension stores the following data in your browser only, using Chrome’s built-in storage APIs:

  • Cached summaries — Generated summaries are cached locally for 24 hours to avoid redundant API calls. Stored in chrome.storage.local.
  • Monthly usage count — A counter tracking how many summaries you have generated in the current billing month, used to enforce the free tier limit. Stored in chrome.storage.local.
  • Compare mode list — Listings you have saved for side-by-side comparison. Stored in chrome.storage.local.
  • Onboarding state — Whether you have seen the first-time welcome overlay. Stored in chrome.storage.local.
  • Anthropic API key (optional) — If you choose to provide your own Anthropic API key, it is stored in chrome.storage.sync and synced across your signed-in Chrome instances via Google’s own encrypted infrastructure. It is never sent to our servers.

All locally stored data can be removed by uninstalling the extension.

Third-Party Services

When generating a summary, the extension communicates with the following external services. No personal information (name, email, account details) is ever included in these requests.

Service Data Sent Purpose
StaySummary API
(Cloudflare Workers)		 Review text, listing title, nightly price, neighborhood data, browser language Backend proxy that forwards data to Anthropic’s Claude API and returns the AI-generated summary. No data is persisted on this server.
Anthropic Claude API Review text, listing title, nightly price, neighborhood context, browser language AI-powered summarization of reviews and neighborhood analysis. Governed by Anthropic’s Privacy Policy.
OpenStreetMap Overpass API Listing latitude & longitude Fetches nearby points of interest (restaurants, transit, shops, safety infrastructure) for the neighborhood analysis tab. This is a public, free API.
Nominatim (OpenStreetMap) Listing latitude & longitude Reverse geocoding to determine the listing’s country and US state (required for crime data lookups). This is a public, free API.
FBI Crime Data API
(api.usa.gov)		 US state abbreviation Fetches publicly available state-level crime statistics for US listings only, used in the safety score. This is a public US government API.
Airbnb Public API Listing ID (from page URL) Fetches publicly available reviews for Airbnb listings. No user credentials or account data are used.
ExtensionPay / Stripe Payment details (handled entirely by Stripe) Subscription payment processing for the premium plan. We never have access to your payment card details. Governed by Stripe’s Privacy Policy.

Tracking, Analytics & Cookies

StaySummary does not use tracking scripts, analytics services, cookies, or any form of behavioral monitoring. We do not sell, share, or monetize your data in any way.

Your Rights & Control

Since all personal data is stored locally in your browser, you have full control at all times:

  • Remove all extension data by uninstalling StaySummary from Chrome.
  • Clear cached summaries through Chrome’s extension storage settings.
  • Revoke your Anthropic API key at any time through your Anthropic account.
